Core Purpose

Two questions answered before every submission

VERIFIQ is built around the two checks every BIM coordinator must confirm before uploading to CORENET-X or NBeS.

1
Is the IFC model complete?

Is every element populated with all the data that Singapore CORENET-X (IFC+SG 2025) and Malaysia NBeS (UBBL 1984) require? VERIFIQ checks entity classes, classifications, all Pset_ and SGPset_ property sets, property values, data types, enumerations, spatial containment, georeferencing (SVY21/GDM2000), GUID uniqueness, and geometry validity for all 8 Singapore agencies simultaneously.

2
Is the classification chain complete?

When a classification code is present on an element (for example, A-WAL-EXW-01 for an external wall), are all the related SGPset_ property sets and required property values also present? VERIFIQ checks the full classification-to-property chain for all 128 embedded IFC+SG codes, per agency, per CORENET-X gateway.

Workflow

From IFC to CORENET-X in 5 steps

VERIFIQ fits into your existing BIM authoring workflow. No new accounts, no cloud dependencies.

1
Export IFC from your BIM tool

ArchiCAD: use the IFC+SG Export Translator from info.corenet.gov.sg with IFC4 Reference View. Revit: use the IFC+SG Shared Parameters and export settings. Export as .ifc, .ifczip, or .ifcxml.

ArchiCAD · Revit · Tekla · OpenBIM
2
Open the IFC file in VERIFIQ

Click Open IFC File. VERIFIQ parses the STEP file natively: no xBIM, no cloud upload, fully offline. The 3D model appears immediately in the viewer.

100% offline · No account required
3
Click Check All, Singapore

One click runs all 20 IFC data levels, all 128 classification code checks, and all 89 Singapore design code rules simultaneously. The Director's Report is generated automatically.

169+ checks · All 8 agencies · Director's Report
4
Fix issues with the Property Editor

Click Fix on any Critical or Error finding that has a property. Enter the correct value and click Apply Fixes. VERIFIQ saves a corrected IFC file. Classifications and entity classes must still be fixed in your BIM tool.

IFC Property Editor · _VERIFIQ_FIXED
5
Export report and submit

Export a Word or PDF compliance report, BCF file for your BIM coordination team, and an Excel findings workbook in one click. Submit the corrected IFC file to CORENET-X with confidence.

Word · PDF · Excel · BCF · JSON · CSV · XML
