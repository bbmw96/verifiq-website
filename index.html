LevelCheckCoverage
1IFC Entity ClassValidates class against IFC4 schema. Flags proxies with AI-suggested replacement.

3ObjectTypeWhen PredefinedType=USERDEFINED, checks ObjectType is populated.
4Classification ReferenceMandatory IfcClassificationReference (IFC+SG 2025 / NBeS 2024).
5Classification EditionReferences current edition: IFC+SG 2025 / NBeS 2024.
6Mandatory Pset_All standard IFC4 property sets per element type.
7SGPset_ (Singapore)CORENET-X agency-specific property sets (SGPset_WallFireRating etc.).
8Property ValuesEvery required property is populated with a valid value.
9Data TypesBOOLEAN/REAL/INTEGER/STRING types per IFC4 schema.
10Enumeration ValuesAgainst permitted lists: space categories, fire ratings, gateway values.
11Spatial ContainmentEvery element assigned to an IfcBuildingStorey.
12Storey ElevationsNo duplicate elevations within and across discipline files.
13GeoreferencingSingapore: SVY21/EPSG:3414 mandatory. Malaysia: GDM2000 recommended.
14Site HierarchyIfcProject → IfcSite → IfcBuilding → IfcBuildingStorey chain.
15GUID UniquenessEvery GlobalId unique across all loaded discipline files.
16MaterialsStructural and fire-rated elements have material specifications.

19IFC Schema VersionCORENET-X requires IFC4 Reference View ADD2 TC1.

Architecture

Built on solid foundations

6 separate C# projects, 14,000+ lines, fully offline — no cloud dependency anywhere in the stack.

🖥 WPF + WebView2 Shell
C# .NET 8 WPF desktop shell with Microsoft WebView2 for the compliance dashboard, 3D viewer, and results UI.
🔍 IFC STEP Parser
Native C# IFC parser. Handles Brep, ExtrudedAreaSolid, and MappedItem geometry. Extracts all entities, properties, and placements.
⚙️ Rules Engine
89 Singapore agency rules, 30 Malaysia UBBL rules, 40 SG design code rules, 28 MY design code rules, 51 property requirements.
🧊 Three.js 3D Viewer
Web-ifc WASM geometry engine (primary) with C# mesh data fallback. Elements colour-coded by compliance status. Click-to-inspect.
📤 Report Generator
8 export templates × 8 formats: Word, PDF, Excel, CSV, JSON, XML, Markdown, BCF. All generated locally using DocumentFormat.OpenXml and ClosedXML.
🔐 Security Module
1,001 SHA-256 licence keys across 5 tiers. Application integrity checking. No activation server — all validation is offline and perpetual.
